# Closet Capsule Designer — Specification

## Goal
Design a minimalist capsule wardrobe from someone's existing clothing, maximizing outfit combinations while identifying gaps worth filling — no wasteful shopping sprees.

## Target Audience
People drowning in clothes but feeling like they have "nothing to wear," looking to simplify their wardrobe without starting from scratch.

## Requirements
1. Analyze the provided clothing inventory and select 30-40 core pieces that form the capsule
2. Organize selections by category: tops, bottoms, layers, dresses/one-pieces, shoes, accessories
3. Generate at least 25 distinct outfit combinations from the capsule pieces
4. Include a color palette analysis showing which colors dominate and how they coordinate
5. Identify 3-5 "gap pieces" that would significantly expand outfit combinations if purchased
6. Flag pieces from the existing wardrobe that don't coordinate with anything (declutter candidates)
7. Provide seasonal adaptation notes — how the capsule shifts for warm vs cold weather
8. Include a "power outfit" recommendation for interviews, dates, and casual Fridays
9. Consider fabric care requirements — don't pair dry-clean-only with wash-and-wear if lifestyle doesn't support it

## Output Format
Color palette overview → Capsule piece list (by category) → Outfit combinations grid → Gap analysis → Declutter candidates → Seasonal notes

## Quality Criteria
- Outfits are genuinely wearable for the person's stated lifestyle
- Color coordination is visually coherent, not just theoretically compatible
- Gap pieces are specific (not "a nice blazer" but "a navy cotton blazer")
- Declutter reasoning is honest but kind
- Mix of everyday, professional, and going-out combinations
